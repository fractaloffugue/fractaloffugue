Alphacore Research

Social Media Analytics Course Project

TL;DR

Analyzed 18 million Ethereum token transactions to reveal critical liquidity concentration in centralized exchanges (CEXs), exposing systemic market risks.

Key Findings

  • Power-law distribution: Small number of CEX wallets control disproportionate liquidity share
  • Centralization risk: Binance addresses show hub-and-spoke structure with single points of failure
  • DEX resilience: Decentralized exchanges exhibit more distributed influence patterns

Technical Approach

  • Network graph construction with 6M+ nodes
  • Louvain modularity maximization (Q>0.74)
  • PageRank centrality analysis
  • Fruchterman-Reingold force-directed layouts

Tools & Technologies

Python: NetworkX, pandas, NumPy
Visualization: Gephi, Matplotlib
Data: Chartalist ERC-20 dataset

SmartAdmit

Machine Learning Course Project

TL;DR

Developed an automated college admission decision system using multi-layer perceptron with multi-modal features, achieving 72% accuracy for admission predictions on 12,000 applications.

Key Results

  • 58% overall accuracy: Balanced performance across admit/deny/waitlist classes
  • 72% admission accuracy: Strong performance on positive admission decisions
  • Feature importance: Word clouds revealed key application terms driving decisions

Technical Approach

  • Multi-layer perceptron with 2 hidden layers (100 neurons each)
  • TF-IDF vectorization for textual features
  • K-means clustering and PCA for dimensionality reduction
  • Ensemble methods combining numerical, categorical, and text data

Dataset & Tools

Data: 12,000 undergraduate applications
Features: Numerical (GPA, test scores), categorical (demographics), textual (essays)
Tools: Python, scikit-learn, pandas, matplotlib
